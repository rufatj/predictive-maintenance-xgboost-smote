# Advanced Predictive Maintenance: Optimizing Failure Detection via XGBoost & SMOTE

##  Overview
Industrial machine failures cause millions of dollars in downtime annually. This project provides an advanced data science solution to predict equipment breakdowns using real-time sensor metrics (temperatures, rotational speed, torque)  and while the baseline machine learning model achieved a high raw accuracy, it suffered from severe **class imbalance**—failing to intercept 19 critical breakdowns. To solve this real-world production challenge I implemented **SMOTE (Synthetic Minority Over-sampling Technique)** to artificially balance the dataset dramatically increasing the models sensitivity to real failures.




Python, XGBoost, Scikit-learn
